# TaskManager

Gestor de tareas simple en Java

## Descripción
TaskManager es una aplicación de escritorio desarrollada en Java que permite gestionar tus tareas diarias de forma sencilla desde la consola. Con ella puedes crear nuevas tareas, visualizar la lista completa, marcar las que vayas completando y eliminar las que ya no sean necesarias.

Este proyecto está orientado a practicar y demostrar conocimientos de programación orientada a objetos (POO) en Java, uso de colecciones y menús interactivos en consola.

## Funcionalidades principales
- Añadir tareas con título, descripción y fecha límite.
- Consultar la lista de todas las tareas y ver cuáles están pendientes y cuáles completadas.
- Marcar tareas como realizadas.
- Eliminar tareas fácilmente por su posición en la lista.

## Tecnologías y herramientas utilizadas
- Java (JDK 8 o superior)
- Programación orientada a objetos
- ArrayList para la gestión de datos en memoria
